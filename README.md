# Student Performance Analysis

Exploratory data analysis on a student exam scores dataset using Python, pandas, and seaborn.

## Key Findings
- Male students scored higher in math on average; female students scored higher in reading and writing.
- Students who completed the test preparation course scored higher across all subjects.
- Students with standard lunch scored significantly higher (10+ points) across all subjects compared to those with free/reduced lunch, suggesting a link between socioeconomic factors and academic performance.

## Tools Used
Python, pandas, seaborn, matplotlib, Google Colab
